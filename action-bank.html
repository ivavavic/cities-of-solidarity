The Action Bank.

A growing repository of municipal action ideas for refugee inclusion — colour-coded by sector, filterable by city and readiness. It starts with the 12 priority actions of the Comrat Local Action Plan and the simplified example actions developed for the Bălți LAP workshop, which any municipality can adapt, combine or replace.

Loading the Action Bank…

How to use this bank: statuses run from concept (an idea to adapt) through pilot to investment-ready (specified in a Local Action Plan with owner, cost and timeline). All content loads from data/action_bank.csv — adding an action is one new row in the master Excel file; see the README for the two-minute workflow.

Cities of Solidarity supports Moldovan municipalities to lead on refugee inclusion while strengthening services for all residents — aligning humanitarian, development and municipal systems.
